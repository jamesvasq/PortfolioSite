Utilizing Bootstrap5

I made use of the Bootstrap framework to create this site. It assisted me to easily create a responsive and visually appealing design. Bootstrap provides me with a range of components, such as navigation bars, buttons, and forms, which I'm able to customize to fit the style and theme of my website. A reason I'm choosing to continue learning Bootstrap is the easy and intuitive process it offers that allows my website to look great on both desktop and mobile devices and integrate well with my own CSS. Bootstrap is also a very popular framework, so I was able to find a lot of resources online to help me with any issues I ran into. Lastly, Bootstrap seems to be an industry standard, so I believe it will be a useful skill to have in the future.
